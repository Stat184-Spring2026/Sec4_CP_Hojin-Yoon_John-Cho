# Generative AI Usage Documentation

## Tool Information

- Tool: ChatGPT
- Model: GPT-5
- Provider: OpenAI
- Dates Used: May 2026

---

## How Generative AI Was Used

Generative AI tools were used to assist with:
- Improving code organization
- Understanding Git and GitHub workflows
- Formatting tables and figures
- Improving README structure and wording
- Improving report organization and section formatting

Generative AI was not used to generate raw datasets or fabricate results. All interpretations, analysis decisions, visualizations, and conclusions were reviewed and verified by the authors.

---

## Example Prompts

Example prompts included:
- "How should I organize a data science GitHub repository?"
- "How do I combine multiple QMD files into a final report?"
- "How can I improve reproducibility in this project?"
- "How do I format tables for both HTML and PDF output?"

---

## Verification

All AI-generated suggestions were manually reviewed, edited, tested, and validated before inclusion in the final project.
