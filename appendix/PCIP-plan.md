# PCIP Plan

## Project Goal

The goal of this project is to explore global electric vehicle adoption trends using exploratory data analysis in R. The project investigates EV sales, EV stock, charging infrastructure, and EV-related policy activity from the early 2010s through 2025.

---

## Plan

We planned to:
1. Import and clean EV datasets
2. Filter the data to focus on global historical trends
3. Compare EV sales and EV stock over time
4. Analyze the relationship between EV stock and charging stations
5. Investigate EV-related legislation trends
6. Create visualizations and tables to communicate findings clearly

---

## Create

We created:
- Data wrangling workflows using the Tidyverse
- Summary tables for EV adoption trends
- Line graphs for EV sales and EV charging ratios
- Bar charts for EV-related policy announcements
- A combined Quarto report with cross-references and reproducible code

---

## Improve

Throughout the project, we improved:
- Code readability through comments and organization
- Visualization clarity through labeling and formatting
- Reproducibility through folder structure and relative file paths
- Report structure by combining separate sections into one final report

We also revised plots and tables multiple times to improve readability and interpretation.

---

## Process Reflection

Using the PCIP system helped organize both the coding workflow and the report-writing process. Breaking the project into planning, creation, and improvement stages made collaboration easier and helped ensure the final report was more reproducible and professional.
